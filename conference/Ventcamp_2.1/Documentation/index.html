A) HTML Structure - top

This theme is a responsive one page layout with one column made on Bootstrap 3.0 foundation so please refer to original docs for more info (http://getbootstrap.com/css/). If you would like to place or make a full screen width element - please place it outside of this div. The general template structure is the same throughout the template.

The template consists of:

  1. Ventcamp_HTML/index.html - default main page
  2. Ventcamp_HTML/index-countdown.html - main page with countdown times
  3. Ventcamp_HTML/index-video.html - main page with YouTube video background
  4. Ventcamp_HTML/index-html5-video.html - main page with HTML5 hosted video background
  5. Ventcamp_HTML/index-overlay.html - main page with color overlay and image background
  6. Ventcamp_HTML/index-tickets-email.html - default main pagethat sends an email/tickets upon completion
  7. Ventcamp_HTML/index-tickets-eventbright.html - default main page with EventBright integration
  8. Ventcamp_HTML/index-tickets-paypal.html - default main page with PayPal integration
  9. Ventcamp_HTML/map-settings.json - json file for Advanced Google Map from contacts, used in html
  10. Ventcamp_HTML/process-form.php - form emails processor, send mail()
  11. Ventcamp_HTML/mailchimp-api.php - MailChimp subscription processor
  12. Ventcamp_HTML/registration-form.html - registration form separately
  13. Ventcamp_HTML/icons-reference.html - icon reference
  14. Ventcamp_HTML/coming-soon.html - coming soon page
  15. Ventcamp_HTML/elements/* - theme elements
  16. Ventcamp_HTML/assets/* - theme assets like CSS,JS and images
  17. Ventcamp_HTML/assets/css/style.css - theme main styles
  18. Ventcamp_HTML/assets/less - theme LESS files (for advanced users)
  19. Ventcamp_HTML/assets/Gruntfile.js - theme Grunt file (for advanced users)
  20. Ventcamp_HTML/assets/package.json - theme packages (for advanced users)

All of the boxed blocks within the main content area are nested within a div with an class of "container" Here is the general structure.

Ventcamp is build on LESS and all less files are located inside assets/less directory, the proper way to change the theme base color is with these but this wold require additional knowledge LESS tutorial

B) CSS Files and Structure - top

This template is built on Bootstrap 3.0 foundation and uses several CSS files:

  1. bootstrap.css - the main bootstrap styles
  2. bootstrap-theme.css - several bootstrap amendments to bootstrap
  3. style.css - template styles

While you can surely edit any file you want, please note that main template styles reside in assets/css/style.css file. D) Registration page - top

The only thing you need to do to setup the registration form is to change this file:

  1. HTML/process-form.php - add your email address, edit fields
  2. HTML/registration-form.html - edit form fields + edit JavaScript found on the same page below

The form is validated and then submitted which then sends the data to process-form.php file that just gets the variables and forards the data to your email

E) Coming soon page - top

The only thing you need to do to setup the mailchimp form is to change this file:

  1. Ventcamp_HTML/coming-soon.html - add your date at line #137 and it's ready!

F) Mailchimp form - top

The only thing you need to do to setup the mailchimp form is to change this file:

  1. Ventcamp_HTML/mailchim-api.php - (add Mailchimp API key at line #36 and subscription list ID at line #40)
